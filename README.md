Usage
-----

```xml
<ba.rubicon.widget.battery.BatteryMeterView
            android:layout_width="200dp"
            android:id="@+id/battery_view"
            android:layout_height="90dp"
            app:battery_level="30"
            android:layout_gravity="center"
            app:charging="true"/>
```
